# FinalProjectDefense
Интернет-магазин DublicatAgro — это платформа, которая упрощает закупку запчастей для сельхозтехники. Проект разрабатывала с нуля, отвечая за все этапы: от структуры до интерактивных элементов.
Сайт выполнен на базовых веб-технологиях: HTML, CSS, JavaScript.
Главная страница сайта «DublicatAgro» разработана для привлечения клиентов, заинтересованных в покупке запчастей для сельхозтехники. Её ключевая задача — быстро донести до пользователя ценность сервиса, показать ассортимент и основные преимущества компании.
Страница ориентирована на фермеров, агрохозяйства и специалистов, обслуживающих сельхозтехнику. Дизайн и контент адаптированы под потребности этой аудитории: акцент на надёжности, скорости и поддержке.
