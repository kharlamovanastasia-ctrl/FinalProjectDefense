# FinalProjectDefense
Представляю Вашему вниманию интернет-магазин DublicatAgro — это платформа, которая упрощает закупку запчастей для сельхозтехники. Проект разрабатывала с нуля, отвечая за все этапы: от структуры до интерактивных элементов. С коммерческой точки зрения, интернет-магазин агрозапчастей  привлекателен по следующим причинам:
    • высокий средний чек: одна запчасть может стоить десятки тысяч рублей;
    • возможность выстраивать долгосрочные отношения с корпоративными клиентами;
    • относительно низкие переменные издержки на онлайн-продажи по сравнению с офлайн - магазинами.
      Сайт выполнен на базовых веб-технологиях: HTML, CSS, JavaScript, Bootstrap, LocalStorage.
